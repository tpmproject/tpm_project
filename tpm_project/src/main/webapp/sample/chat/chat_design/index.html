Quick start

1.- Include the files 

    
  
    <!-- Bootstrap and JQuery -->
    
    <script src="jquery.min.js"> </script>
    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css">    
    
    <!-- Plugin -->
    <link rel="stylesheet" type="text/css" href="chatSocketAchex.css">
    <script src="chatSocketAchex.js"></script>
2.- Create HTML Element 

   <div id="Elchat"></div>
3.- Call the the plugin on a container as your wish 

   $('#Elchat').ChatSocket();
4.- All done!

Options

you can custom by passing options when call the plugin, all available options are listed below. call the the plugin on a container wish you wish 

        $('#Elchat').ChatSocket({
                    option: value,
                    option2: value2
       });
